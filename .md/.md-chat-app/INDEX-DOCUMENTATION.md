## 📝 Conventions de la documentation

### Icônes utilisées
```
✅ = Complété / OK
❌ = Non complété / KO
⚠️ = Attention / Warning
🆕 = Nouveau
🔧 = À configurer
📱 = Mobile
🖥️ = Backend
🗄️ = Base de données
📚 = Documentation
🧪 = Test
⚡ = Rapide
```

### Structure des documents
```
# Titre principal
## Section
### Sous-section
#### Détail

Code blocks : ```language
Warnings : > ⚠️ Attention
Info boxes : > 💡 Astuce
```
